<선형회귀의 장단점>
장점:
- 계산 비용이 낮아 대규모 데이터에서도 빠르게 적용이 가능함.
- 각 독립변수가 종속변수에 미치는 영향을 명확히 설명할 수 있음.
- 통계적으로 유의미한 결과를 얻어내기에 적합하며, 데이터의 선형적 관계를 분석하는 데에 유용함.

단점: 
- 독립변수와 종속변수 간의 관계가 비선형이면 부적합함.
- 이상치에 민감하여 결과를 왜곡할 수 있음.
- 비선형성이나 고차원의 복잡한 데이터에 대해 예측력이 낮음.

<K-최근접 이웃의 장단점>
장점:
- 데이터의 분포에 대한 가정이 없어 비선형적인 데이터도 잘 처리함.
- 분류와 회귀에 모두 사용 가능함.
- 특정 데이터의 지역적 패턴에 민감하므로, 복잡한 데이터 분포를 잘 학습할 수 있음.

단점:
- 예측 시 모든 데이터 포인트와 거리를 계산하므로 데이터가 많아질수록 속도가 느려짐.
- 많은 데이터가 필요하며 고차원 데이터에서는 성능이 저하될 수 있음.
- 모든 특징이 동일한 가중치를 가져 특징별로 가중치를 설정하지 않으면 거리 계산에 왜곡이 생길 수 있음.

<GDP 계산 결과 비교>
37655를 입력값으로 설정했는데 선형회귀 모델은 결과값을 5.88로 출력하였고 K-최근접 이웃 모델은 6.0으로 출력하였다.
학습 데이터 상으로 봤을 때 37655가 입력값이면 결과값으로는 약 5.65가 나와야 하므로 선형회귀가 비교적 정답에 근접한 것을 확인할 수 있었다.
이번 과제의 학습 데이터를 보면 선형적 데이터라는 것을 알 수 있는데 확실히 선형적 데이터에는 선형회귀를 적용하는 것이 효과적이라는 것을 알 수 있었다.
